# Rick & Morty Flutter App

A **Flutter** application that lets users explore Rick & Morty characters with features like:

- Infinite scrolling
- Search & filters
- Favorites management
- Detailed character views
- Offline mode
- Error handling



### **Architecture**


- **Clean Architecture** with clear separation into:
  - Core
  - Features
  - Data
  - Domain
  - Presentation
